# game_tracker

A new Flutter project.

This app also demonstrates fetching game descriptions from multiple
sources. If RAWG does not provide a Russian description, the code will
attempt to use IGDB (Twitch) instead. To enable IGDB lookups add
`IGDB_CLIENT_ID` and `IGDB_TOKEN` to `assets/.env`.
